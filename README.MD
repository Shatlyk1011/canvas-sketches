## Installation

1. **Clone this repository:**

   ```bash
   git clone https://github.com/Shatlyk1011/canvas-sketches
   ```

2. **Install canvas-sketch globally**
   ```
   npm i canvas-sketch-cli -g
   ```

3. **Install dependencies**
  ```
   cd canvas-sketches && npm i
  ```
